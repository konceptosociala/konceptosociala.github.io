Flatbox: Simplicity & Rust

Flatbox is an open source engine, which supports PBR rendering, physics, Lua scripting and much more
Get started Documentation GitHub

Features

PBR rendering

Flatbox provides modern PBR renderer written with modern OpenGL.


Physics

Fast 3D physics simulations driven by Rapier3D physics engine.


Extension system

Apply custom extensions to the application for custom rendering or logic pipeline


Lua scripting

Write your runtime scripts with fast and elegant Lua language
